# WinterBlox ❄ FPS LAUNCHER 1.2

This program is designed to help people improve their Roblox performance. WinterBlox is an optimization program that configures Windows services and screen resolution. When you open WinterBlox alongside Roblox, the screen resolution will automatically be set to 800x600 when you close the program, an automatic resolution of 1366 x 768 is applied (you can change this manually in your Windows display settings). When you open the program, several Windows services will also be automatically disabled; they will be restored when Roblox is closed with the program still open.

When you close the program, an automatic resolution of 1366 x 768 is applied (you can change this manually in your Windows display settings). When you open the program, several Windows services will also be automatically disabled; they will be restored when Roblox is closed with the program still open.

This new version brings changes such as a new program called "WinterBloxServices". This disables Windows services when Roblox is opened and they are restored normally when it is closed. There are also new aesthetic details, and I will continue working on this.
